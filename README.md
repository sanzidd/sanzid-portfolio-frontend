## ✨ Features

- **Responsive Design** — Fully optimized for desktop, tablet, and mobile devic
- **Dark / Light Mode** — Toggle between themes for comfortable viewing
- **Multi-page Layout** — Dedicated pages for About, Education, Skills, Projects, and Contact
- **Admin Dashboard** — Backend-connected admin panel for managing content
- **CV Download** — Direct PDF download and view functionality
- **Project Showcase** — Featured projects with tech stack tags and descriptions
- **Contact Section** — Easy way for collaborators and recruiters to reach out
- **Smooth Animations** — Polished UI with scroll effects and transitions

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser. No build tools or dependencies required.

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sanzidd/sanzid-portfolio-frontend.git

# 2. Navigate into the project directory
cd sanzid-portfolio-frontend

# 3. Open index.html in your browser
# You can simply double-click index.html, or use a live server extension
```

> **Tip:** If you use VS Code, install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension and click **"Go Live"** for the best development experience.
